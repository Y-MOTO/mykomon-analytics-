# 日本の人事制度の変遷と会計事務所・税理士法人の実態

**調査日：2026年4月26日**

---

## 1. 日本の人事制度 歴史的変遷

### 戦後〜高度経済成長期（1945〜1970年代）：三本柱の確立

戦後の日本型雇用は「終身雇用・年功序列賃金・新卒一括採用」の三本柱で構築された。

- 1946年：電力系組合が「電産型賃金体系」を提案。生活保障給＋勤続給が全体の7割を占め、これが年功序列賃金の源流となった
- 高度経済成長期：企業が社員を「メンバー」として囲い込むメンバーシップ型雇用が定着。長期雇用が企業と社員の双方にとって合理的だった
- 企業内訓練（OJT）・内部昇進・単一労働組合が三本柱を支える構造として機能

### バブル期（1980年代）：制度の膨張

- 完全雇用状態が続き、採用競争が激化
- 福利厚生・退職金・住宅手当など間接的処遇が拡充
- 年功給の運用が「生活費補填」から「慣例的昇給」へと形骸化し始める

### バブル崩壊後（1990年代）：成果主義の導入と失敗

- 1990年代後半、富士通・ソニーなど大企業が「成果主義人事制度」を一斉導入
- 目標管理制度（MBO）と連動し、個人の短期成果で報酬を決定する仕組みへ
- **失敗のパターン（日本における成果主義の教訓）**
  - 単年度成果で評価 → 2〜3年かかる開発・育成行動が評価されなくなる
  - 後輩指導・ナレッジ共有が「損な行動」になる（フジテック・日本マクドナルド）
  - 数値化できない仕事（信頼構築・顧客維持）が軽視される
  - 評価の納得感がなく、不満が蓄積して離職が増加

### 2000年代：制度の揺り戻しと多様化

- 成果主義の弊害が顕在化した企業が「役割等級制度」へ回帰・移行
- 「コンピテンシー評価」「360度評価」の普及
- 非正規雇用の拡大が社会問題化

### 2010年代：女性活躍・働き方改革

- 2015年：女性活躍推進法施行
- 2019年：働き方改革関連法（残業規制・同一労働同一賃金）
- 大企業を中心に「副業解禁」「フレックスタイム」が広がる

### 2020年代：ジョブ型への本格移行（現在進行中）

- 富士通・パナソニック・日立・NEC・NTTなど大手が相次いでジョブ型人事を導入
- **2024年8月：政府が「ジョブ型人事指針」を公表**（内閣官房・経済産業省・厚生労働省）
  - 職務ごとに求めるスキルを明確化し、リ・スキリング・自律的なキャリア形成を促進
  - 職能給（能力）から職務給（仕事内容）への転換を国家的に推奨
- **2025年の主流トレンド**：「ジョブ型＋成長支援型」のハイブリッド。急激な移行ではなく段階的な制度設計が主流

## 2. 会計事務所・税理士法人の人事制度の実態

### 業界の基本構造

- 2025年10月時点：税理士登録者数 **82,114人**、税理士法人届出数 **8,269事務所**
- 大多数が中小規模（職員10〜50名）のオーナー経営型事務所
- 大規模法人（Big4・準大手）と中小事務所で人事制度の成熟度に大きな格差がある

### 規模別の人事制度の実態

#### ■ Big4・大手税理士法人

| 項目 | 内容 |
|---|---|
| 職階 | Staff → Senior → Manager → Senior Manager → Director → Partner |
| 給与 | マネージャーで年収1,000万円前後、パートナーで1,500万円〜数千万円 |
| 評価 | 360度評価・コンピテンシー評価・目標管理（OKR導入も増加） |
| 育成 | 体系的な研修制度・資格取得支援 |
| 新卒 | 初年度年収400万円〜。3〜4年でシニアアソシエイトへ昇格 |
| ボーナス | 月給の4〜5ヶ月分 |

#### ■ 中小税理士法人・会計事務所（職員10〜50名）

| 項目 | 内容 |
|---|---|
| 給与水準 | 平均年収517〜547万円（全国平均）。未経験者20〜25万円/月から |
| 給与決定 | **所長の主観的判断**が多い。体系的評価制度が未整備な事務所が多数 |
| 職階 | 明文化されていないケースが多い。「スタッフ→シニア→マネージャー」が暗黙の序列 |
| 評価軸 | 資格保有（税理士・公認会計士・社労士）・経験年数・担当先数・顧客満足度 |
| ボーナス | 月給の2〜3ヶ月分が標準 |
| 昇給 | 年功色が強く、明確な評価基準なしに「なんとなく上がる」事務所が多い |

### 中小会計事務所に特有の人事課題

#### 1. 評価制度の未整備
- 「公平な評価基準がない」と感じる職員が多く、不満・離職につながる
- 所長への依存度が高く、人事判断が恣意的になりやすい
- **有資格者と無資格者の処遇差が不透明**で、有資格者の不満が蓄積しやすい

#### 2. キャリアパスの不透明感
- 「どうすれば昇格・昇給するのか」が明文化されていない
- 特に若手・Z世代は「成長できる見通し」が見えないと早期離職する
- 税理士試験勉強中の職員は「合格後の処遇が不明」なまま在籍する

#### 3. 業務の属人化
- 担当者と顧問先の1対1の関係が固定化し、ナレッジが共有されない
- コア人材が退職すると顧問先ごと失うリスク（バス係数＝1問題）
- 日報・業務記録が不十分で業績・工数の把握が困難

#### 4. 採用競争の激化
- 税理士試験合格者数が減少傾向で、有資格者の採用競争が激しい
- 大手法人・一般企業経理職との処遇競争で中小事務所が不利になりやすい
- リモートワーク・フレックス対応が進んでいない事務所は若手から敬遠される

#### 5. 成果主義導入の難しさ
- 顧問料が固定制のため、売上への個人貢献を測定しにくい
- 「顧客満足度」「応答速度」「追加業務の獲得」など間接指標での評価が必要
- 日報・業務ログなしでは評価の根拠が作れない

## 3. 最新トレンドと業界の方向性（2024〜2025年）

### 中小事務所でも始まりつつある変化

- **職階制の明文化**：Tier型の職階（スタッフ→シニア→マネージャー）を制度化する事務所が増加
- **ジョブディスクリプション（JD）の整備**：採用・評価・育成を一貫させる基盤として注目
- **資格手当の体系化**：税理士・公認会計士・社労士への専門職プレミアムを明文化
- **集合型インセンティブ**：事務所全体の目標達成で全員にボーナスを支給するモデル（個人競争を避ける）
- **業務可視化ツールの導入**：日報システム・工数管理ツールを評価根拠として活用する動きが萌芽

### 政府・業界団体の動向

- 2024年：日本税理士会連合会が「働き方改革・人材育成」をテーマとした事務所経営指針を更新
- ジョブ型人事指針（2024年8月政府公表）は中小専門職事務所にも適用を推奨
- 税理士法改正議論：法人化・業務範囲拡大が進み、組織経営への移行が加速

## 4. 御社（55名・税理士法人）への示唆

| 課題 | 業界の実態 | 御社のアプローチ |
|---|---|---|
| 評価基準が不透明 | 中小事務所の大半が同じ問題を抱える | 日報データを評価根拠として数値化（Phase 1の日報システムが直結） |
| キャリアパス不明 | 若手離職の最大要因 | Tier1〜4の職階を明文化し全員に説明（Phase 2） |
| 属人化リスク | 業界全体の構造問題 | 日報タグによる業務可視化でナレッジを組織資産に |
| 有資格者の不満 | 処遇差が不透明な事務所が多い | 専門職プレミアム（10〜15%）を明文化（Phase 3） |
| 採用競争 | 中小は大手・一般企業と競合 | 透明な評価制度・ジョブ型が「選ばれる事務所」の差別化要因になる |

**結論：御社が取り組む「日報可視化→評価根拠の数値化→ジョブ型移行」は、業界の最先端モデルと一致している。中小税理士法人でこの変革を完遂できれば、採用力・定着率・顧問先品質の三点で圧倒的な優位性を持つ。**
