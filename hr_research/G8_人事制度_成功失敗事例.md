# G8各国の人事制度と世界の事例

## 成功事例

### Buurtzorg（オランダ）
- 看護師14,000人・管理職ゼロ。10〜12人小チームが完全自律
- 成功の鍵：客観的指標（患者満足度）が全員に見えていた
- 会計事務所への示唆：担当者別の詰まり率・完了率が「見える」日報システムがボトムアップの前提

### Morning Star（米国）
- 世界最大トマト加工会社。同僚合意書（CLOU）で自律と説明責任を両立
- 会計事務所への示唆：「誰が何に責任を持つか」を担当者間で合意する文化が先

### Big4モデル（グローバル会計事務所）
- 年次評価廃止 → 継続フィードバック。OKRで目標の透明化
- 会計事務所への示唆：年1回の評価より月次・週次のフィードバックが定着につながる

## 失敗事例

### フジテック（1993年）
- 完全成果主義を導入 → 失敗を恐れ革新が消滅 → ヒット製品が生まれなくなった
- 教訓：個人成果主義は短期的には機能するが、協調・育成・挑戦を阻害する

### 日本マクドナルド（2006年）
- 成果主義導入 → 後輩育成が消滅（評価されないから） → 2012年に部分回帰
- 教訓：育成・指導行動を評価しない制度は、組織の再生産力を壊す

## 各国の特徴（概要）

| 国 | 特徴 |
|---|---|
| 米国 | 職務給＋OKR＋継続フィードバック（Big4は年次評価廃止） |
| 英国 | パートナーシップ制・専門家の自律性を尊重 |
| ドイツ | 職務記述書・労使協議・透明な賃金体系 |
| イタリア | 4階層分類（階層が上がるほど自律度が増す） |
| 日本 | 大企業先行でジョブ型移行中・中小専門職事務所はこれから |
