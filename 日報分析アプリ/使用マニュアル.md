# MyKomon 日報分析アプリ 使用マニュアル

---

## 概要

MyKomonの日報データを集計・グラフ表示するWebアプリです。  
所長がブラウザで操作し、Claude AI（Sonnet 4.6）による経営コメントも生成できます。

**分析対象**: 全日報（拡張機能でタグを付けたものに加え、自由記述のみの日報も含む）

---

## 2つの起動方法

### A. クラウド版（推奨）

インターネット上で動作するため、**自分のPCがオフでもスマホや別PCからアクセス可能**です。

`クラウド版を開く.bat` をダブルクリックするか、以下のURLをブラウザで開きます。

> **URL**: https://vkxoqm2dmyx6tcorcsaoaz.streamlit.app

---

### B. ローカル版（このPCでのみ使用）

**`起動.bat`** をダブルクリックします。数秒後ブラウザが自動で開きます。

```
http://localhost:8501
```

> アプリを終了するには、起動時に開いた黒いコマンドウィンドウを閉じてください。

#### 初回のみ：セットアップ

**`セットアップ.bat`** をダブルクリックして完了を待ちます（数分・要インターネット接続）。

**APIキーの初回設定（IT管理者が所長PCで実施）：**

1. `【管理者用・DROPBOX管理不可】APIキー暗号化.bat` を所長のPCでダブルクリック
2. API キー（`sk-ant-` で始まる文字列）を貼り付けて Enter
3. 完了後、このBATファイルを所長のPCから削除する

設定後は所長がキーを扱う必要は一切ありません。アプリ起動時に自動入力されます。

**環境変数が消えた場合（PCリセット後など）：**

`APIキー設定.bat` をダブルクリックするだけで復元できます（キー入力不要）。

## 画面の見方（サイドバー）

アプリ左側のサイドバーで以下を設定します。

### 🔐 MyKomon認証

| 項目 | 内容 |
|------|------|
| MyKomon ID | MyKomonのログインID |
| MyKomon パスワード | MyKomonのログインパスワード |

> 入力した認証情報は画面表示専用で、サーバーには保存されません。

---

### 📥 データ取得

取得したい期間（開始年月・終了年月）を選択し **「📥 CSVをエクスポート実行」** をクリックします。

処理の流れ：

1. MyKomonに自動ログイン
2. 指定期間のCSVを月ごとにダウンロード
3. 完了後、自動でデータ読み込みとグラフ表示

> エクスポートには月あたり10〜30秒かかります。複数月を指定すると時間がかかります。

---

### 📂 分析対象（既存CSVを使う場合）

すでにCSVがローカルに保存されている場合は、以下で直接読み込めます。

| 項目 | 内容 |
|------|------|
| CSVフォルダ | CSVが保存されているフォルダのパス |
| Anthropic API キー | AI分析機能を使う場合（省略可） |
| 🔄 既存CSVを再読み込み | ボタンクリックでCSVを読み込み |

---

## データの読み込み状況

データ読み込み後、画面上部に以下が表示されます：

```
📅 読み込み期間：2026年1月 〜 2026年4月　｜　全 8,046 件　｜　タグ付き 312 件
```

- **全N件**：全日報件数（タグなし日報を含む）
- **タグ付きN件**：Edge拡張機能で入力した構造化タグが付いた日報件数

## タブ一覧

### 👤 タブ1　担当者別分析

**目的**: 誰が何件対応し、誰が業務で詰まっているかを把握する

#### 上段：担当者別 日報件数（全件）
全日報（タグなし含む）を担当者ごとに集計した棒グラフです。

#### 中段：担当者別 詰まり件数（タグ付きのみ）
ステータスが「継続」または「中断」の日報件数を担当者ごとに集計します。  
タグ付き日報が蓄積されるまでは表示されません。

#### 下段：阻害要因内訳
担当者を選択すると阻害要因の内訳を横棒グラフで表示します。  
「（全員）」を選択すると全社の阻害要因を一覧できます。

---

### 🏢 タブ2　顧問先別分析

**目的**: どの顧問先への対応が多く、どこで問題が起きているかを把握する

#### 上段：顧問先別 対応件数（全件）
全日報を顧問先ごとに集計します。

#### 下段：顧問先別 詰まり件数（タグ付きのみ）
棒の色がオレンジ〜赤になるほど「Red（緊急）」の日報が多い顧問先です。

---

### 🔄 タブ3　業務×工程 ボトルネック（タグ付きのみ）

**目的**: どの業務区分・工程でよく詰まるかを把握する

縦軸が工程、横軸が業務区分のヒートマップです。  
色が濃い（赤い）マスほど詰まりが多いボトルネックです。

> タグ付き日報が蓄積されると表示されます。

---

### 📅 タブ4　月次トレンド

**目的**: 業務量の推移・詰まりが改善しているかを確認する

#### 上段：月別 日報件数（全件）
全日報の月ごとの件数推移を棒グラフで表示します。

#### 下段：月次ステータス推移（タグ付きのみ）
完了／継続／中断の月別推移と詰まり率（%）を表示します。  
「継続」「中断」が増えていれば業務負荷が高まっているサインです。

---

### 🤖 タブ5　AI分析レポート

**目的**: 集計結果をClaude AIに渡し、経営者向けの文章コメントを生成する

#### 使い方

1. サイドバーに Anthropic API キーを入力
2. **「📝 AIレポートを生成」** をクリック
3. 30〜60秒待つとレポートが表示される

#### レポートの内容

- 経営者への要約
- 要注意事項（優先順位付き箇条書き）
- 推奨アクション（担当者名・顧問先名を含む具体的な指示）
- **全日報統計 + タグ付き統計の両方を考慮した分析**（タグなし日報のメモも40件サンプルとして参照）

#### フォローアップ質問

レポート生成後、下部の入力欄に追加質問を入力して「送信」を押すと会話形式で深掘りできます。

例：
- 「田中さんへの具体的な声かけ方法を教えてください」
- 「○○株式会社の対応を優先すべき理由を詳しく」
- 「来月の所内MTGで話し合うべき議題を3つ挙げてください」

> APIキーは使用量に応じて課金されます。1回の分析レポート生成あたり目安 **数円〜十数円** 程度です。

## よくある問題

### 「タグ付き日報はまだありません」と表示される

Edge拡張機能（MyKomon 日報入力補助）を使って記入された日報がまだないか、  
データ期間が古すぎる可能性があります。  
全件ベースの活動量グラフは引き続き表示されます。

### グラフに担当者名・顧問先名が表示されない

MyKomonのCSVのカラム名が想定と異なる場合があります。  
CSVをExcelで開き、担当者・顧問先・日付の列名を確認してください。  
確認した列名を開発担当者に連絡し、`parser.py` の候補リストに追加してもらってください。

### CSVエクスポートが失敗する（「ログイン失敗」エラー）

MyKomon IDとパスワードが正しいか確認してください。  
MyKomonのパスワードを最近変更した場合は、サイドバーの入力値を更新してください。

### AI分析で「エラー」が表示される

- APIキーが正しく入力されているか確認してください（`sk-ant-` で始まる）
- Anthropicのアカウントに残高があるか確認してください

### ブラウザが開かない・アプリが起動しない（ローカル版）

コマンドプロンプトで以下を実行してください：

```
cd /d "（日報分析アプリフォルダのパス）"
python -m streamlit run app.py
```
