# MyKomon 日報分析アプリ 使用マニュアル

---

## 概要

MyKomonからエクスポートしたCSVを読み込み、職員の業務状況を自動集計・グラフ表示するWebアプリです。  
所長がブラウザで操作し、Claude AI（Sonnet 4.6）による経営コメントも生成できます。

---

## 初回セットアップ（1回だけ）

### 手順1　ライブラリのインストール

`日報分析アプリ` フォルダ内の **`セットアップ.bat`** をダブルクリックします。

```
完了しました。「起動.bat」をダブルクリックしてアプリを起動してください。
```

上記メッセージが表示されたらインストール完了です。ウィンドウを閉じてください。

> **注意**: インターネット接続が必要です。初回のみ数分かかります。

---

### 手順2　Anthropic APIキーの取得（AI分析機能を使う場合のみ）

AI分析レポート機能を使うには、Anthropic社のAPIキーが必要です。

1. [https://console.anthropic.com](https://console.anthropic.com) にアクセスしてアカウントを作成
2. 「API Keys」メニューから新しいキーを作成
3. `sk-ant-...` で始まるキーをコピーしておく

> APIキーは使用量に応じて課金されます。1回の分析レポート生成あたり目安 **数円〜十数円** 程度です。

---

## 毎回の起動方法

**`起動.bat`** をダブルクリックします。

数秒後、ブラウザが自動で開き以下のURLが表示されます。

```
http://localhost:8501
```

ブラウザが自動で開かない場合は、上記URLを手動でアドレスバーに入力してください。

> アプリを終了するには、起動時に開いた黒いコマンドウィンドウを閉じてください。

---

## 画面の見方

### サイドバー（左側）

アプリ起動後、まずサイドバーで設定を行います。

| 項目 | 内容 |
|------|------|
| **CSVフォルダ** | MyKomonのCSVが保存されているフォルダのパス |
| **Anthropic APIキー** | AI分析機能を使う場合に入力（省略可） |
| **データ読み込みボタン** | クリックするとCSVを読み込み、グラフが表示されます |

#### CSVフォルダの確認方法

`csv-export` フォルダ内の **`config.json`** を開き、`save_dir` の値を確認してください。

```json
{
  "save_dir": "C:\\Users\\motoz\\Desktop\\日報CSV"
}
```

この値をそのままサイドバーの「CSVフォルダ」に貼り付けてください。

### タブ一覧

データ読み込み後、5つのタブで分析結果を確認できます。

---

## タブ1　担当者別分析

**目的**: 誰が業務で詰まっているかを把握する

### 上段：担当者別 詰まり件数

ステータスが「継続」または「中断」の日報件数を担当者ごとに集計した棒グラフです。  
件数が多い担当者ほど業務が滞っています。

### 下段：阻害要因内訳

担当者を選択すると、その人の阻害要因の内訳を横棒グラフで表示します。  
「（全員）」を選択すると全社の阻害要因を一覧できます。

---

## タブ2　顧問先別分析

**目的**: どの顧問先で問題が起きているかを把握する

顧問先ごとの詰まり件数を棒グラフで表示します。  
棒の色がオレンジ〜赤になるほど「Red（緊急）」の日報が多い顧問先です。

> 顧問先列がCSVに含まれていない場合は「顧問先列が見つかりません」と表示されます。

---

## タブ3　業務×工程 ボトルネック

**目的**: どの業務区分・工程でよく詰まるかを把握する

縦軸が工程（例：作業中・チェック中・レビュー待ち）、横軸が業務区分（例：監査・申告）のヒートマップです。  
色が濃い（赤い）マスほど詰まりが多いボトルネックです。

> 工程タグ（【工程】）が付いた日報が存在しない場合はデータなしと表示されます。

---

## タブ4　月次トレンド

**目的**: 業務の滞り具合が改善しているかを確認する

### 上段：ステータス推移（折れ線グラフ）

月ごとの「完了」「継続」「中断」件数の推移を表示します。  
「継続」「中断」が増えていれば業務負荷が高まっているサインです。

### 下段：詰まり率（棒グラフ）

全日報に占める「継続+中断」の割合（%）を月ごとに表示します。

> 日付列がCSVに含まれていない場合は「日付列が見つかりません」と表示されます。

---

## タブ5　AI分析レポート

**目的**: 集計結果をClaude AIに渡し、経営者向けの文章コメントを生成する

### 使い方

1. サイドバーに Anthropic API キーを入力していることを確認
2. **「AIレポートを生成」** ボタンをクリック
3. 30〜60秒待つとレポートが表示される

### レポートの内容

- 経営者への要約（3〜5行）
- 要注意事項（優先順位付き箇条書き）
- 推奨アクション（担当者名・顧問先名を含む具体的な指示）

### フォローアップ質問

レポート生成後、下部の入力欄に追加質問を入力して「送信」を押すと、  
会話形式でさらに詳しく聞くことができます。

例：
- 「田中さんへの具体的な声かけ方法を教えてください」
- 「○○株式会社の対応を優先すべき理由を詳しく」
- 「来月の所内MTGで話し合うべき議題を3つ挙げてください」

## よくある問題

### データ読み込みで「タグ付き日報が見つかりません」と表示される

CSVの日報テキスト欄に `【業務区分】` などの構造化タグが含まれていない可能性があります。  
Edge拡張機能（MyKomon 日報入力補助）を使って記入された日報のみが集計対象です。

### グラフに担当者名・顧問先名が表示されない

MyKomonのCSVのカラム名が想定と異なる場合があります。  
CSVをExcelで開き、担当者・顧問先・日付の列名を確認してください。  
確認した列名を開発担当者に連絡し、`parser.py` の候補リストに追加してもらってください。

### AI分析で「エラー」が表示される

- APIキーが正しく入力されているか確認してください（`sk-ant-` で始まる）
- Anthropicのアカウントに残高があるか確認してください

### ブラウザが開かない・アプリが起動しない

1. `起動.bat` を右クリック →「管理者として実行」を試してください
2. それでも起動しない場合は、コマンドプロンプトで以下を実行してください：
   ```
   cd /d "（日報分析アプリフォルダのパス）"
   python -m streamlit run app.py
   ```

## データの更新方法

### 手動でCSVを更新する場合

1. MyKomonからCSVをエクスポートし、`config.json` の `save_dir` フォルダに保存
2. アプリのサイドバーで「データ読み込み」をクリック

### 自動更新（毎月1日・Windowsタスクスケジューラ）

`csv-export` フォルダの `setup.bat` でセットアップ済みの場合、  
毎月1日AM9:00に自動でCSVがエクスポートされます。  
アプリ起動時に「データ読み込み」を押せば最新データが反映されます。
