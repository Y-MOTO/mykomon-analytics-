# MyKomon 日報CSV自動エクスポート 使用マニュアル

---

## 概要

MyKomonの日報CSVを自動でダウンロードし、指定フォルダに保存するツールです。  
毎月1日AM9:00にWindowsタスクスケジューラで自動実行することができます。  
手動でいつでも実行することも可能です。

---

## ファイル構成

```
csv-export/
├── config.json          設定ファイル（ユーザー名・パスワード・保存先）
├── mykomon_export.py    エクスポート本体（Pythonスクリプト）
├── setup.bat            初回セットアップ用（1回だけ実行）
├── run.bat              手動実行用（毎回使う）
├── task_scheduler.xml   タスクスケジューラ定義（setup.batが自動登録）
└── 使用マニュアル.md    このファイル
```

## 初回セットアップ（1回だけ）

### 前提条件

- Python 3.10以上がインストールされていること  
  → インストール済みか不明な場合：コマンドプロンプトで `python --version` を実行して確認  
  → 未インストールの場合：[https://www.python.org/](https://www.python.org/) からダウンロード

### 手順1　setup.bat を実行

`setup.bat` を**右クリック → 「管理者として実行」**します。

> 管理者権限が必要なのはタスクスケジューラ登録のためです。

以下の処理が自動で行われます：

1. 必要なPythonライブラリをインストール（`playwright`, `python-dateutil`）
2. Microsoft Edge の自動操作用ドライバをセットアップ
3. `config.json` がメモ帳で自動的に開く

### 手順2　config.json を設定

メモ帳が自動で開きます。以下の4項目を設定してください。

```json
{
  "username": "ここにMyKomonのログインID",
  "password": "ここにMyKomonのパスワード",
  "save_dir": "C:\\Users\\motoz\\Desktop\\日報CSV",
  "months_back": 1
}
```

| 項目 | 内容 |
|------|------|
| `username` | MyKomonのログインID |
| `password` | MyKomonのパスワード |
| `save_dir` | CSVの保存先フォルダパス（`\\` で区切ること） |
| `months_back` | 何ヶ月前のデータを取得するか（通常は `1`）|

> **セキュリティについて**: config.json はこのフォルダ内にのみ保存されます。他のシステムには送信されません。

設定が終わったらメモ帳を保存して閉じてください。

### 手順3　タスクスケジューラ登録

setup.bat の最後に以下の確認メッセージが表示されます。

```
タスクスケジューラに毎月1日 AM9:00 の自動実行を登録しますか？ (y/n):
```

- 自動実行を登録する場合 → `y` と入力してEnter
- 手動実行のみでよい場合 → `n` と入力してEnter

```
セットアップ完了。
手動実行する場合は run.bat をダブルクリックしてください。
```

上記が表示されたらセットアップ完了です。

## 手動でCSVを取得する

**`run.bat`** をダブルクリックします。

Edgeブラウザが自動で起動し、以下の処理が順番に実行されます：

1. MyKomonにログイン
2. エクスポートページへ移動
3. 対象月（config.jsonの `months_back` 月前）を選択
4. CSVをダウンロード
5. `save_dir` で指定したフォルダに `日報_YYYYMM.csv` という名前で保存

完了すると以下が表示されます：

```
保存完了: C:\Users\motoz\Desktop\日報CSV\日報_202504.csv
完了しました。
```

> 実行中はEdgeブラウザが操作される様子が表示されます。操作中はブラウザを触らないでください。

## 自動実行（タスクスケジューラ）

setup.bat でタスクスケジューラを登録した場合、**毎月1日AM9:00**に自動でCSV取得が実行されます。

### 登録内容の確認

1. スタートメニューで「タスクスケジューラ」を検索して開く
2. 「タスクスケジューラライブラリ」の中に **「MyKomon日報CSV取得」** があれば登録済み

### 自動実行の条件

- PCの電源が入っていること
- MyKomonにアクセスできるネットワークに接続していること

> PCがスリープ中の場合は実行されません。その場合は手動で `run.bat` を実行してください。

### タスクの手動削除

自動実行が不要になった場合は、タスクスケジューラを開き  
「MyKomon日報CSV取得」を右クリック → 「削除」で削除できます。

## 取得されるCSVについて

### ファイル名

```
日報_YYYYMM.csv  （例: 日報_202504.csv）
```

### エクスポート設定（自動で行われる内容）

| 設定項目 | 値 |
|----------|----|
| 期間 | 月単位（config.jsonの months_back 月前） |
| 職員 | すべて |
| 顧問先 | 未選択（＝全顧客が出力される） |
| 業務分類 | すべて |

## 設定変更方法

### 保存先フォルダを変更したい

`config.json` の `save_dir` を書き換えて保存してください。  
フォルダが存在しない場合は自動で作成されます。

### 取得月を変更したい

`config.json` の `months_back` を変更してください。

| 値 | 取得対象 |
|----|---------|
| `1` | 先月（デフォルト） |
| `2` | 2ヶ月前 |
| `0` | 今月 |

### パスワードを変更した場合

`config.json` の `password` を新しいパスワードに書き換えて保存してください。

## よくある問題

### 「エラー: config.json にユーザー名とパスワードを設定してください」

`config.json` の `username` または `password` が空のままです。  
config.json をメモ帳で開いて設定してください。

### 「エラー: ログインフォームが見つかりません」

MyKomonのログイン画面のHTML構造が変わった可能性があります。  
開発担当者に連絡し、`mykomon_export.py` のセレクタ調整を依頼してください。

### エクスポートボタンが押せず止まる

MyKomonのエクスポートページのHTML構造が変わった可能性があります。  
同様に開発担当者へ連絡してください。

### CSVが保存されているか確認したい

`config.json` の `save_dir` で指定したフォルダをエクスプローラーで開き、  
`日報_YYYYMM.csv` ファイルが存在するか確認してください。

### タスクスケジューラの登録に失敗した

管理者権限が必要です。`setup.bat` を右クリック → 「管理者として実行」で再実行してください。

## 日報分析アプリとの連携

取得したCSVは、同フォルダの `config.json` と `save_dir` を共有している  
**日報分析アプリ**（`../日報分析アプリ/`）から自動的に読み込まれます。

分析アプリの使い方は `../日報分析アプリ/使用マニュアル.md` を参照してください。
