## プロジェクト概要

税理士事務所で使用している業務管理システム「MyKomon（名南経営製）」の日報入力を補助するシステム群を開発。
MyKomonにはAPIが存在しないため、以下の3フェーズに分けて開発した。

| フェーズ | 内容 |
|---------|------|
| **フェーズ1** | Microsoft Edge 拡張機能（日報入力時のカード型フローティングUI） |
| **フェーズ2** | CSV自動エクスポート（MyKomonからのデータ取得） |
| **フェーズ3** | 日報分析Webアプリ（Streamlit Cloud + ローカル版） |

## システム全体の構成（2026-04-27 完成時点）

### フェーズ1：Edge拡張機能

```
edge-extension/
├── manifest.json   # Manifest V3・host: https://*.mykomon.com/*
├── popup.html      # カード型UIのHTML
├── popup.js        # カードUIのロジック（未使用・content.jsに統合済み）
├── content.js      # フローティングUIの本体（world: MAIN必須）
└── icons/
```

**主な機能:**
- 画面右上に固定表示されるカード型フローティングUI
- 選択項目：業務区分（5択）・ステータス（3択）・工程（5択）・阻害要因（8択）・緊急度（3択）・工数（5択）
- 継続/中断選択時のみ工程・阻害要因・緊急度を表示
- 「平文に反映する」でTinyMCEエディタ（`tinymce.get('inc_editor')`）へ構造化タグを自動挿入
- タグなし登録インターセプト：警告ダイアログ表示

**技術的に重要なポイント:**
- `content.js` は `"world": "MAIN"` 指定が必須（TinyMCEにアクセスするため）
- テキスト挿入は `tinymce.get('inc_editor').execCommand('mceInsertContent', ...)` で実現
- MyKomonのCSPがインラインスクリプト注入をブロックするため、scriptタグ注入は不可

---

### フェーズ2：CSV自動エクスポート

```
csv_export/
└── mykomon_export_http.py   # requests + BeautifulSoupによるHTTPリクエスト版
```

- ログインURL: `https://www.mykomon.com/MyKomon/login.do`（POST）
- エクスポートURL: `https://www.mykomon.com/groupware/downloadScheduleView?equipmentFlag=0`
- Playwright廃止→HTTPリクエスト版に移行（Streamlit Cloud互換）

### フェーズ3：日報分析Webアプリ

**Tab構成（6タブ）:**

| タブ | 内容 |
|------|------|
| Tab1 👤 担当者別分析 | 日報件数・タグ付き率・詰まり分析 |
| Tab2 🏢 顧問先別分析 | 対応件数・詰まり件数 |
| Tab3 🔄 業務×工程 ボトルネック | ヒートマップ |
| Tab4 📅 月次トレンド | ステータス推移・詰まり率 |

**ファイル構成と用途:**

| ファイル | 用途 |

| `日報分析アプリ/app.py` | ローカル版（バッチファイルから起動）※クラウドとは非同期 |
| `配布用/所長配布用/日報分析アプリ/` | 手動同期コピー（修正のたびに更新が必要） |

## 重要な設計・運用ルール

### タグ命名規則（変更禁止）
過去データとの一貫性を保つため、以下は運用開始後に変更しない。

```
【業務区分】申告 / 相談 / 監査 / 記帳代行 / その他
【ステータス】完了 / 継続 / 中断
【阻害要因】資料待ち / 確認中 / 知識不足 / 客先都合 / その他
#業務区分_{値} #ステータス_{値} #阻害要因_{値}
```

### ファイル更新時の同期ルール
`app.py` 等を更新した際は以下3箇所に手動で同期する：

```
1. app.py（root）         ← Streamlit Cloud用
2. 日報分析アプリ/app.py   ← ローカル版用
3. 配布用/所長配布用/日報分析アプリ/  ← 所長配布用（手動コピー）
```

### クラウド版編集の注意
**必ずルートの `app.py` を直接編集する。**  
`日報分析アプリ/app.py` を編集してもStreamlit Cloudには反映されない。
